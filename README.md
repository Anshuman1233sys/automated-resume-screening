# Automated Resume Screening System

## 📌 Project Overview
This project is a simple implementation of an Automated Resume Screening System. It helps in filtering and ranking resumes based on skills and experience. The system reduces manual effort in the hiring process and provides quick results.

## 🚀 Features
- Upload resume (basic UI)
- Extract skills from resume (dummy logic)
- Analyze experience
- Rank candidates based on matching criteria
- Display results in sorted order

## 🛠️ Technologies Used
- Python (for backend logic)
- HTML, CSS, JavaScript (for UI)
- Basic data structures for processing

## 📂 Project Structure

│── main.py

## ⚙️ How It Works
1. User uploads resume
2. System extracts skills and experience
3. Matches with required job criteria
4. Calculates score
5. Displays ranked candidates

## 📊 Output
The system shows shortlisted candidates with ranking based on their skills and experience.
